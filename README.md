# Campus Navigation System

A **Nile University Campus Navigation System** that models the campus as an
**undirected, unweighted graph** and demonstrates core graph-representation techniques
and **Breadth-First Search (BFS)** traversal. Built for the *Discrete Mathematics*
course (Spring 2026).

## What it does

- Represents the campus three ways: **Adjacency List**, **Adjacency Matrix**, and **Edge List**.
- Demonstrates step-by-step **Adjacency List → Adjacency Matrix** conversion.
- Runs a **BFS traversal** from any starting building (level-by-level exploration).
- Interactive text menu: view each representation, graph statistics, search buildings, and run BFS.

## Run it

Requires **Python 3.6+** (standard library only — no external packages).

```bash
cd code
python campus_navigation.py
```

Then use the numbered menu (1–8) to explore the features.
